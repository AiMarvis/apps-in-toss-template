# 시즈널 소재 캘린더

시기, 요일, 시간대에 따라 유저 반응이 달라진다. 토스는 금융 앱이므로, 급여일·명절·결제일 등 금전 관련 시기가 특히 중요하다.

---

## 월별 마케팅 시즈널

| 월 | 시즌 키워드 | 소재 활용 팁 | 예시 타이틀 |
|----|-----------|------------|-----------|
| 1월 | 새해, 목표, 신년운세 | 새해 결심 + 습관 앱 연계 | "새해 첫 도전" |
| 2월 | 설날, 세뱃돈, 발렌타인 | 보너스·세뱃돈 테마 + 관계 | "설날 운세 도착" |
| 3월 | 개학, 봄, 벚꽃 | 새 시작 + 일상 루틴 | "봄맞이 습관" |
| 4월 | 벚꽃, 야외활동 | 가벼운 재미 앵글 | "4월 미션 열림" |
| 5월 | 어린이날, 어버이날, 가정의달 | 선물·감사 앵글 | "감사 메시지 도전" |
| 6월 | 여름 시작, 방학 준비 | 게임·재미 앵글 강화 | "여름 게임 도전" |
| 7-8월 | 여름방학, 휴가 | 여가·시간 여유 앵글 | "방학 특별 미션" |
| 9월 | 추석, 개학 | 명절 보너스 + 새 시작 | "추석 보너스 9원" |
| 10월 | 가을, 핼러윈 | 캐릭터·재미 + 시즌 이벤트 | "핼러윈 미션" |
| 11월 | 수능, 블프, 11절 | 시험 끝 해방감 + 소비 | "수능 끝 게임 도전" |
| 12월 | 크리스마스, 연말 정산 | 연말 회고 + 선물 + 금전 | "올해 마지막 보상" |

## 주간 패턴

### 요일별 유저 심리와 소재 앵글

| 요일 | 유저 심리 | 추천 앵글 | 예시 |
|------|---------|---------|------|
| 월요일 | 새 시작, 루틴 의지 | 일상루틴, 행동묘사 | "이번 주 습관 시작" |
| 화~수 | 일상, 집중 | 행동묘사, 확실한보상 | "3분이면 끝나요" |
| 목요일 | 주말 기대, 여유 | 호기심, 감성·재미 | "판다가 뭘 하고 있을까" |
| 금요일 | 해방감, 재미 추구 | 감성·재미, 게임 | "주말 게임 준비됐어요" |
| 토~일 | 여유, 탐색, 소비 | 호기심, 확실한보상 | "주말 보너스 도전" |

### 주말 vs 평일 전략

- **평일**: 짧고 실용적인 메시지 (행동묘사, 루틴)
- **주말**: 여유롭고 탐색적인 메시지 (호기심, 재미)

## 시간대 패턴

> ⚠️ 스마트 발송은 시스템이 자동으로 최적 시간에 발송한다. 직접 시간을 지정하는 것이 아니라, 소재 톤을 시간대에 맞게 준비하면 된다.

### 시간대별 소재 톤

| 시간대 | 유저 상태 | 소재 톤 | 예시 |
|--------|---------|--------|------|
| 아침 (7~9시) | 출근 준비, 체크 | "아침 루틴" 톤 | "아침 체크 한 번" |
| 오전 (9~12시) | 업무 중, 짧은 휴식 | "가볍게 한 번" 톤 | "버튼 3번이면 끝" |
| 점심 (12~13시) | 여유, 탐색 | "재미있는 거 해볼까" 톤 | "점심시간 퀴즈" |
| 오후 (14~17시) | 집중, 피로 | "잠깐 쉬면서" 톤 | "3분이면 끝나요" |
| 저녁 (18~21시) | 퇴근, 여유 | "오늘 보상 받기" 톤 | "퇴근 후 9원" |
| 밤 (21~23시) | 휴식, 게임 | "하루 마무리" 톤 | "오늘 마지막 기회" |

## 급여일 전후 패턴

토스는 금융 앱이므로 급여일(주로 매월 25일 전후) 전후 유저의 금전 감수성이 달라진다.

| 시기 | 유저 심리 | 소재 전략 |
|------|---------|---------|
| 급여일 -3일 | "돈이 다 떨어졌다" | 소액 보상 강조 ("9원이라도") |
| 급여일 당일 | "돈이 들어왔다" 여유 | 새 도전 유도 ("이번 달 도전") |
| 급여일 +3일 | 소비 기분, 여유 | 재미·탐색 앵글 |
| 월말 | "이번 달 안 받은 거 없나" | 미수령 보상 강조 ("안 받은 27원") |

## 이벤트 캘린더 템플릿

파트너가 자신의 앱에 맞게 작성하는 연간 캘린더.

```
[시즌 소재 캘린더 — OOO 앱]

| 시기 | 이벤트 | 소재 앵글 | 검토 제출일 | 활성화일 |
|------|--------|---------|----------|---------|
| 1/1 | 새해 이벤트 | 새해 결심 + 보상 | 12/23 (연말 전) | 1/1 |
| 2/4 | 설날 이벤트 | 운세 + 세뱃돈 | 1/28 | 2/4 |
| ... | ... | ... | ... | ... |

※ 검토 제출일: 활성화 1주 전 (영업일 기준)
※ 반려 시 재제출 2~3일 추가
```

## 주의사항

1. **날씨·계절 표현은 보수적으로**: "오늘 비 온다" 같은 실시간 정보는 낙관적으로 틀릴 수 있다. "봄이 왔어요" 같은 넓은 시즌 표현은 괜찮다.
2. **공휴일은 미리 체크**: 검토가 영업일 기준이므로, 공휴일 연휴 전에 미리 제출한다.
3. **시기별 소재는 별도 캠페인으로**: 이벤트 소재와 상시 소재를 같은 캠페인에 섞지 않는다. MAB 최적화가 꼬인다.
