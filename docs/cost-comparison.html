🔒 mTLS 프록시

Firebase Cloud Functions vs Vercel Serverless
토스 OAuth mTLS 교환 전용. Firebase CF는 호출당 과금(프록시 용도 거의 무료). Vercel은 좌석당 $20+/월 고정 — 프록시 하나 때문에 비쌈.
Firebase CF: ~$0 Vercel: $20+/월

👤 인증 (Auth)

Firebase Auth vs Supabase Auth
50K MAU까지 둘 다 무료. 500K+ MAU에서 Firebase가 ~23% 저렴 ($0.0025 vs $0.00325/MAU). Firebase는 Cloud Functions와 같은 생태계라 토큰 연동이 자연스러움.
Firebase: $0.0025/MAU Supabase: $0.00325/MAU

⚡ 백엔드 (DB+API)

Convex vs Supabase
Convex는 실시간+API+Cron이 기본 내장. Supabase는 Pro $25 + Compute $100+ 필수. Realtime은 별도 $2.5/1M 메시지.
Convex Pro: $25/dev Supabase 실질: $125+/월
💡
핵심 차이: 현재 스택은 Firebase 생태계(Cloud Functions + Auth)에서 mTLS 프록시와 인증을 원스톱 처리합니다. 대안 스택에서는 Vercel(프록시)과 Supabase(인증)을 크로스 서비스 연동해야 하며, Vercel 좌석 비용이 추가됩니다.

숨은 비용과 함정

가격표에 나오지 않는 실제 운영 비용

⚠️

Supabase: "Pro $25" 의 진실

헤드라인 가격은 $25/월이지만, 프로덕션에서는 Dedicated Compute가 필수입니다. Shared CPU(기본)로는 트래픽 피크 시 쿼리 타임아웃이 발생합니다. 실질 최소 비용: $125/월 (Pro $25 + Small Compute $100).

⚠️

Supabase 무료 티어: 7일 비활성 중단

무료 프로젝트는 7일간 API 호출이 없으면 자동으로 일시 중단됩니다. 개발 단계에서도 주기적으로 접근해야 유지됩니다. Convex 무료 티어에는 이런 제한이 없습니다.

⚠️

Vercel: 토스 미니앱에서 불필요한 비용

토스 미니앱은 ait deploy로 Granite 인프라에 직접 배포됩니다. Vercel 호스팅은 완전히 불필요한 중복 비용입니다. CDN, Edge Network, 빌드 머신 비용 모두 낭비.

⚠️

Supabase Realtime: 연결 수 과금

Supabase Realtime은 메시지당 과금 + 피크 동시 연결 수 과금($10/1,000 connections). Convex는 실시간 구독이 기본 포함이며, 동시 세션 1,000개(무료) ~ 10,000개(Pro)까지 추가 비용 없음.

⚠️

Vercel Serverless: Cold Start 지연

Serverless 함수는 사용하지 않으면 종료되어 첫 요청에 1~3초 지연이 발생합니다. Convex 함수는 항상 warm 상태를 유지하여 일관된 <50ms 응답을 보장합니다.

토스 미니앱 특화 이점

이 아키텍처가 토스 플랫폼에서 특히 유리한 이유

📱 왜 Convex + Firebase가 토스 미니앱에 최적인가?

토스 미니앱의 특수한 요구사항에 정확히 맞는 아키텍처

🏠

호스팅 = Granite

ait deploy로 배포 완료. Vercel, Netlify, CloudFront 전부 불필요. 배포 비용 $0.

🔒

mTLS 인증 = Firebase

토스 OAuth mTLS 프록시는 Cloud Functions로 구현. Supabase Edge Functions에서는 mTLS 인증서 관리가 더 복잡.

⚡

실시간 = Convex 기본

useQuery()만으로 실시간 구독. 별도 Realtime 설정이나 WebSocket 관리 불필요.

🛠

심사 호환성

Convex의 단순한 구조는 심사 시 요구되는 연동해제/데이터삭제 API를 쉽게 구현 가능.

🚀

개발 속도

스키마 정의 → 자동 타입 생성 → 프론트에서 바로 사용. API 레이어 작성 불필요.

🛡

보안 기본 제공

SQL Injection 불가능 (SQL 없음). Convex는 TypeScript 쿼리만 사용. OWASP 위험 감소.
