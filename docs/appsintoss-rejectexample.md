## ✅ 출시 전 반려사례 체크리스트

<aside>
🚨

**심사 제출 전 아래 항목을 모두 확인하세요.** 실제 반려 사례에서 추출한 필수 점검 항목입니다.

### 🔙 네비게이션

- [ ]  자체 구현 백버튼/뒤로가기 버튼을 **모두 제거**했는가?
- [ ]  **공통 내비게이션 백버튼**을 통해 히스토리 백이 동작하는가?
- [ ]  최초 화면(온보딩/랜딩 페이지, 예: `/onboarding`)에서 공통 내비게이션 백버튼 터치 시 **미니앱이 종료**되는가? *(리프레쉬만 되고 종료되지 않는 경우 수정 필요)*
- [ ]  기능 스킴 접속 시 공통 내비게이션 백버튼 및 **커스텀 설정 버튼이 정상 동작**하는가?
- [ ]  **공통 내비게이션 앱 아이콘/로고**가 정상 노출되는가?
- [ ]  **자체 햄버거 메뉴를 제거**하고 공통 내비게이션 커스텀을 활용하는가?
- [ ]  공통 내비게이션 **즐겨찾기(하트) 아이콘** 등 커스텀 버튼이 정상 동작하는가?

### 🔐 로그인/인증/권한

- [ ]  토스 로그인은 **인트로 뷰 이후**에 진행되는가? (앱 시작 직후 로그인 화면 노출 금지)
- [ ]  **자체 이용약관을 제거**하고 토스 로그인 플로우에서 약관 동의를 받는가?
- [ ]  토스 로그인 연동 해제 시 **콜백을 통해 로그아웃 처리**가 되는가? ([로그인 개발 문서](https://developers-apps-in-toss.toss.im/login/develop.html))
- [ ]  연동 해제 후 **재로그인 플로우**가 정상 진행되는가? (자체 로그아웃 기능 없이도 동작해야 함)
- [ ]  위치/카메라 등 권한이 필요한 경우 **토스 권한 문서 기반 바텀싯**을 구현했는가?
- [ ]  권한 목록에 표시된 권한이 **실제로 요청**되는가?
- [ ]  권한 거부 시 **재요청 또는 안내 방법**이 있는가? *(개선 권고)*
- [ ]  **탈퇴하기 기능**이 정상 동작하는가?(탈퇴하기 기능을 구현했을 때만, 없으면 skip)

### 🔗 스킴/라우팅

- [ ]  등록한 모든 **기능 스킴 URL이 404 없이 정상 동작**하는가?
- [ ]  메인 스킴 외 **기능 스킴 랜딩 페이지**가 정상 렌더링되는가?

### 🎨 UI/UX

- [ ]  앱 상단에 **자체 헤더**가 없는가? (인앱 브라우저처럼 보이면 안 됨)
- [ ]  바텀싯이 화면 전체를 덮어 **닫기 버튼 터치가 불가능**한 경우가 없는가?
- [ ]  홈 화면에서 **불필요한 좌우 스크롤**이 비활성화되어 있는가?
- [ ]  좋아요/즐겨찾기 등 **기능 동작 후 데이터가 정상 반영**되는가?
- [ ]  횟수 제한 기능(랜덤 뽑기 등)이 **소진 후에도 정상 안내**되는가?
- [ ]  이미지 저장 등 **주요 기능 버튼이 정상 동작**하는가?
- [ ]  화면 요소(배너 등)에 **과도한 깜빡임/애니메이션**이 없는가?
- [ ]  **탭바가 플로팅 형태**로 구현되어 있는가?
- [ ]  탭바 여백 및 아이콘/문구 **크기가 적절**한가?
- [ ]  **확대/축소 핀치줌**이 비활성화되어 있는가?
- [ ]  앱 내 **얼럿이 모두 TDS 모달**로 적용되어 있는가?
- [ ]  **광고 로딩**이 정상적으로 동작하는가?
- [ ]  **재화 소모 기능**이 정상 동작하는가?

### 📝 텍스트/브랜딩

- [ ]  **앱 이름**이 정확하게 입력되어 있는가?
- [ ]  앱 이름이 **정보등록과 동일**하게 노출되는가?
- [ ]  공유하기, 메타 태그 등에서 **브랜드명이 통일**되어 있는가?
- [ ]  **구독 상품**이 포함되어 있지 않은가? (구독 오인 워딩 제거)
- [ ]  UI에 표시된 **결제 금액**과 실제 결제 금액이 일치하는가?

## 📊 반려 유형 분류

| 유형 | 설명 | 빈도 |
| --- | --- | --- |
| 🔙 **네비게이션** | 자체 백버튼, 공통 내비게이션 미활용 | 높음 |
| 🔐 **로그인/인증** | 연동 해제 콜백, 로그아웃/재로그인 처리 | 높음 |
| 🔗 **스킴/라우팅** | 404 오류, 기능 스킴 미동작 | 중간 |
| 🎨 **UI/UX** | 헤더, 바텀싯, 스크롤, 기능 버튼 무반응, 깜빡임 | 중간 |
| 📝 **텍스트/브랜딩** | 앱 이름, 브랜드명 오류 | 낮음 |
